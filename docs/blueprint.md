# **App Name**: College Connect

## Core Features:

- Secure Login: User authentication with pseudo and password for secure access.
- Real-time Messaging: Instant messaging for direct communication between students.
- Group Channels: Dedicated channels for clubs, courses, and study groups.
- File Sharing: Easy sharing of documents, notes, and assignments.
- Notification System: Alerts for new messages, events, and important announcements.
- Profile Customization: Personalized profiles with student information and interests.

## Style Guidelines:

- Primary color: Violet (#800080) for trust and unity.
- Secondary color: Light Gray (#F8F9FA) for a clean interface.
- Accent color: Academic Gold (#FFC107) for important notifications.
- Font: 'Open Sans' for readability and a modern feel.
- Use consistent, simple icons for navigation and features.
- Clean and intuitive layout for easy navigation and information access.
- Subtle animations for smooth transitions and user feedback.
